# Niagara Open Data (niagaraopendata-ca)

Niagara Open Data is a county government open-data portal for Canada running **CKAN**, exposing a standardized catalog API over ~529 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/niagaraopendata-ca/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=niagaraopendata-ca-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.9.9

## API
- **Niagara Open Data CKAN Action API** — base `https://niagaraopendata.ca/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://niagaraopendata.ca)
- Live endpoint verified: `https://niagaraopendata.ca/api/3/action/status_show` · ~529 datasets
